# Shopping List App (쇼핑 리스트 앱)

간단하고 직관적인 쇼핑 리스트 웹 애플리케이션입니다.

## 기능

- 아이템 추가/삭제
- 완료 체크 표시
- 완료된 항목 일괄 삭제
- 로컬 스토리지에 데이터 저장 (브라우저를 닫아도 유지)
- 반응형 디자인

## 사용 방법

1. `index.html` 파일을 브라우저에서 열어주세요.
2. 입력창에 구매할 아이템을 입력하고 "추가" 버튼을 클릭하거나 Enter 키를 누르세요.
3. 체크박스를 클릭하여 완료 표시를 할 수 있습니다.
4. × 버튼으로 개별 항목을 삭제할 수 있습니다.
5. "완료 항목 삭제" 버튼으로 완료된 모든 항목을 한번에 삭제할 수 있습니다.

## 기술 스택

- HTML5
- CSS3 (애니메이션, 그라데이션)
- Vanilla JavaScript
- LocalStorage API
